The tools I reach for, and roughly why.

## Languages

Python for almost everything, with Javascript and SQL where they fit.

## ML & data

PyTorch and TensorFlow for models; scikit-learn for classical work; LangChain for retrieval
pipelines. Pandas and NumPy for data, with Matplotlib, Seaborn, and Plotly for plots.
Sentence-Transformers and a Chroma vector store when a project needs embeddings and
similarity search.

## Visualization

Power BI and Tableau for dashboards; Streamlit when a project just needs a quick interactive
UI.

## Everything else

Git and GitHub for version control, including GitHub Actions for automation, which is what
runs the class-recording pipeline I built at ISL.
