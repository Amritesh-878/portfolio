# Portfolio — Amritesh Praveen

My personal site, built like a documentation site (think [fumadocs](https://fumadocs.dev), except the documented system is me): MDX content with sidebar and Ctrl/⌘ K search, an **AI twin** — a hybrid-RAG chatbot with a visible retrieval trace, grounded in the curated corpus in `content/ai-twin/` — and my published RL game **Hunter Wumpus** ([paper](https://doi.org/10.5281/zenodo.20076630)) playable right in the browser.

**Stack:** TypeScript · Next.js (App Router) · Fumadocs · Tailwind CSS · Gemini API. The site runs on Vercel; the game's FastAPI + PPO backend runs on a Hugging Face Space ([hunter-wumpus](https://github.com/Amritesh-878/hunter-wumpus)).

**Status:** under construction.
